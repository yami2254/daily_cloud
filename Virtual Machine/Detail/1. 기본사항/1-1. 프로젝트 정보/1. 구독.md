# 구독(Subscription)
## 자원 생성시 목적
- 배포하고자 하는 자원의 논리적 위치를 선택하기 위함 
## Tips
- 다른(or 잘못된) 구독에 배포를 했을 경우 Resource Move를 이용해서 구독간 이동이 가능
  -   Resource Move에는 제약 사항이 존재하므로 만능이 아님  

-----
더 상세한 것은 구독(링크)를 참고(추후 수정)
